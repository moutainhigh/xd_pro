第二节 基础教育课程改革
考点
7：基础教育课程改革的实施

课程结构

1. 对课程类型的调整

（1）整体设置九年义务教育课程：小学阶段以综合课程为主。

（2）设置综合实践活动并作为必修课程。

（3）农村中学课程为当地社会经济发展服务。

2. 对科目比重的调整

将下调部分课程比重，并将调整课时分配给综合实践活动和地方与校本课程。

课程内容

课程标准

课程标准体现出普及性、基础性和发展性三个特征。

教材的编写

1. 教材管理由“国编制”转变为“国审制”。

2. 降低知识难度，引进现代信息。

3. 密切联系生活，关注学生个体经验。

4. 重视活动设计，鼓励学生探究创造。

5. 尊重师生个性，给师生广阔的发展空间。

课程实施

1. 学生观

•  学生是发展的人。

•  学生是独特的人。

•  学生是具有独立意识的人。

2. 学习方式

（1）自主学习。

是以学生作为学习的主体，即主动自觉自愿的学习，而不是被动的或不情愿的学习。

（2）合作学习。

是指学生为了完成共同的任务，有明确的责任分工的互助性学习。

（3）探究学习.

倡导学生主动参与，在科学理论指导下，运用科学方法进行问题探究。

3. 教师观

教师角色：合作者、引导者、参与者。

（1）从教学与研究的关系看，教师应教育教学的研究者。

（2）从教学和课程的关系看，教师应是课程的建设者和开发者。

（3）从学校和社区的关系看，教师应是社区型的开放教师。

4. 教师教学行为的转换

（1）在对待师生关系上，强调尊重、赞赏。

（2）在对待教学教学关系上，强调帮助、引导。

（3）在对待自我上，强调反思。

（4）在对待与其他教育者的关系上，强调合作。

课程管理

建立以校为本的教学研究制度、民主科学的教学管理制度、旨在促进教师专业成长的考评制度。

课程评价

学生评价

建立评价学生全面发展的指标体系，重视质性评价方法。

教师评价

建立评价指标体系，强调自评，倡导建立教师、学生、家长、管理者共同参与的多渠道信息反馈的教师评价制度。

课程实施评价

建立以学校评价为基础的评价体系。
