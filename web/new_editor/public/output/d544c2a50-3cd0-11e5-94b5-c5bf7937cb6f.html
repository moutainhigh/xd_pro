第一节 阅读理解能力
考点
1：文字表述的解读

理解重要概念解题技巧

1. 提炼要点

对要求解释概念是什么的题目需要仔细阅读材料，提炼要点。

2. 联系文段背景

有些题目要求结合材料解释概念内涵，一般要解释的概念相对隐晦，需要结合文段背景进行分析，找出内涵。

3. 结合上下文语境

词句理解型题目中，所考查概念的含义常是对其原含义进行引申，或是临时被赋予更深刻的含义。必须联系上下文语境，推敲概念的含义。

4. 代词“朝前找”

有些题目要求解释“这”、“那”等代词指代的内容，做题时只需关注上文。

5. 注意特殊标点

•  冒号破折号具有解释说明的意思

•  分号、顿号常用于引导并列关系

•  双引号具有援引观点、强调、反讽的意思

•  括号用于补充说明

理解重要句子解题技巧

1. 巧用信息、整体把握

根据文段提供的重要信息，如作者、写作时间、文后注释，结合题目要求，揣度文章的大概主旨，再整体把握材料，易形成正确的解题思路。

2. 尊重原文、摘取信息

大多数题目的答案是可以从文中概括提炼的，离开原文反而易出现偏误。

3. 先拟草稿、写全写顺

词句理解题有两个基本要求，一是“踩点”给分，二是文通字顺。多写一般不扣分，答题时尽可能多写点；应先拟草稿，再修改通顺。

4. 语境分析

联系上下文、文段背景，注意特殊标点的提示。

5. 修辞提示法

对于运用了修辞手法的句子，要结合修辞手法的特点理解词句含义，修辞提示法的使用规则为：比喻关键找本体，象征则要看对象。如果理解反语意，褒贬互换要记牢。双关你要想一想，到底它双的是啥？借代种类有很多，抽象意义最常考。对比要比比变化，根据已知推未知。

6. 层次划分法

有两层意思，一是抓句子主谓宾，二是明确句子之间逻辑关系。
