考点
1：感知觉

感觉特性

1. 感觉适应

感觉适应是指刺激持续作用于同一感受器而使感受性发生变化的现象。痛觉是伤害性刺激预警信号。

2. 感觉对比

感觉对比是指对不同性质的刺激作用于同一感受器产生相互作用，使感受性发生变化的现象。

根据刺激呈现时间：同时对比VS继时对比

3. 联觉

联觉是指一种感觉引起另一种感觉的心理现象，是感觉相互作用的体现。

4. 感觉后象

外界刺激停止作用后，暂时保留的感觉印象。分为：正后像，负后像。

知觉特性

1. 知觉选择性

人受自己兴趣和需要等因素的影响，有意或无意地把某些刺激信息或刺激的某些方面作为知觉对象，把其他事物作为背景的现象。

2. 知觉理解性

在知觉过程中，人以自己过去经验解释知觉现象，并用此把它标志出来。

3. 知觉整体性

人不把知觉的对象感知为个别的孤立部分，而把它知觉为一个统一的整体。

4. 知觉恒常性

当知觉的条件在一定范围内改变，知觉的映像仍然保持相对不变。
