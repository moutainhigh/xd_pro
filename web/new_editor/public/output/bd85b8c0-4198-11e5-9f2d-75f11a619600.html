2：终身学习的理念

终身学习概念

终身学习是指社会每个成员为适应社会发展和实现个体发展的需要，贯穿于人的一生的，持续的学习过程。它强调时间的延展性，以一种教育和教育观念存在。

终身学习的内容

1. 学会学习：培养良好学习习惯。

2. 成为学科专家：接受严格的高层次的学科教育。

3. 学习有关教育的学问：在学习专业学科同时掌握有关教育学的学问。

4. 学习信息技术：将现代信息技术转化为现代教学手段。

终身学习的要求

1. 把学习当成生活的习惯，不断学习。

2. 安排并利用一定的时间学习。

3. 将学习与工作和生活相联系，学为所用。

4. 掌握方法和技术，高效率学习。
