第三节 教师观
考点
1：中学教师的专业发展

教师职业的劳动特征

1. 复杂性

（1）教育目的的全面性

（2）教育任务的多样性

（3）劳动对象的差异性

2. 创造性

表现在教师创造性的运用教育教学规律：

（1）因材施教

（2）教学方法上的不断更新

（3）教师需要“教育机智”

3. 主体性和示范性

主体性是指教师自身可以成为活生生的教育因素和具有影响力的榜样。

示范性是指教师的言行举止，包括人品、才能、治学态度等都会成为学生学习的对象。

4. 劳动时间和空间的延展性

劳动时间的延续性是由于教师没有固定的工作时间长度，也没有严格的交接班时间界限。空间的广延性是指教师没有严格界定的劳动场所，这是由影响学生发展因素的多样性决定的。

5. 长期性和间接性

长期性是指人才培养的周期比较长，教育影响具有滞后性。间接性是指教师的劳动不直接创造物质财富，而是以学生为中介实现教师劳动的价值。

6. 个体性与群体性

从劳动手段角度来看，教师的劳动主要是以个体劳动的形式进行。同时教师的劳动成果又是集体劳动和多方影响的结果。

教师专业发展的要求

•  学会学习，成为终身学习者

•  勤于反思，成为反思的实践者

•  恒与研究，成为教育教学的研究者

•  重视沟通，加强交往与合作能力

•  勇于创新，加强创新精神和实践能力

内容

•  建立专业理想

•  形成专业人格

•  拓展和深化专业能力

•  提高专业能力

•  完善专业态度和动机

•  形成专业自我

途径

1. 职前培训

为了有效的结合理论和实践，教师需职前培训中掌握专业技能，形成专业素养。

2. 入职培训

为进一步掌握岗位知识和能力，教师的入职培训在全面了解岗位要求的基础上，需接受老教师的指导。

3. 在职培训

包括到大学或教师进修学院参加培训；参加学术团体与学术活动进行培训；通过教育观摩学习先进的教育理论和实践经验。

4. 自我教育

教师的自我教育就是教师专业自我的主动建构过程。教师反思使教师获得自身专业发展的重要途径，也是最直接最普遍的途径。具体途径如下：

（1）专业阅读。是教师学习的最主要的形式。

（2）网络学习。是随着网络的出现而采用的一种新的学习形式。

（3）校本教研与培训。校本教研与培训是近年来在我国学校出现的一种重要的教师专业发展模式。

（4）行动研究。行动研究是十分有效的和综合的发展途径。

方法

1. 观摩和分析优秀教师的教学活动

对需提高教学经验的新教师进行有计划的组织观摩；对有完备理论知识和洞察力的教师进行非组织化观摩。

2. 开展微格教学

以少数学生为对象，在5~20分钟之内，进行小型课堂教学，同时摄制成录像，课后进行分型，有利于提高训练新教师、提高教学水平。

3. 进行专门训练

训练的关键程序有：每天进行回顾、有意义地呈现新材料、有效的指导课堂作业、布置家庭作业、每周每月进行回顾。

4. 反思教学经验

以教学活动为意识对象，进行认真的评价和调节。

发展的阶段的阶段理论 

分类依据：教师需要和关注焦点的差异 

类型名称：三阶段发展

关注生存阶段 

新教师，关注生存适应性，花费大量时间在人际关系上。 

关注情境阶段 

环境适应后，教师将关注焦点投向了提高学生的成绩，此阶段教师更关注教学过程中的问题。

关注学生阶段 

教师考虑学生的个别差异，能否自觉关注学生是衡量一个教师是否成长成熟的重要标志之一。

分类依据：进入教师职业的发展进程

类型名称：四阶段发展

职前储备阶段

指虚拟关注期，此阶段主体是师范生。

入职调整期

指求生关注期，由师范生向正式教师角色转换，理论与实践的磨合期。

巩固稳定阶段

指任务关注期，教师专业结构等诸方面稳定持续发展的阶段。

创新成熟期

指教师自我专业发展、自我更新时期。

分类依据：教师专业知识和技能的掌握

类型名称：五阶段发展

新手 

获取经验阶段。

熟练新手

理论与实践结合发展阶段。

胜任型 

教师可对所选择的信息做出反应，并承担更多责任。 

精通型 

有更多的教学直觉或领会，可更精准的预测事件。 

专家 

能以非分析性、非随意性的方式，理智地做出反应。
