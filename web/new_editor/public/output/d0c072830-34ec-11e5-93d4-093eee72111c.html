第二节 幼儿游戏特点及指导
考点
2：幼儿游戏的指导

教师指导策略


1. 尊重幼儿自主性

尊重幼儿的意愿和兴趣，尊重游戏氛围，尊重幼儿的想象、探索、表现创造。

2. 以观察为依据

通过观察，了解情况，才能作出合适的指导。

3. 确定指导的方式方法

确定是以自身、以材料还是以幼儿伙伴为媒介进行指导。

4. 确定指导时机

根据幼儿客观需要、主观心态和状况来确定指导时机。

5. 把握指导的对象范围

将重点与一般、个别与集体、局部与整体结合起来，针对具体情景灵活指导。

6. 把握互动节奏

教师应站在幼儿的角度考虑，与幼儿保持节奏一致。

7. 正面评价游戏

正面评价有利于激起幼儿再次游戏的愿望。

教师介入

1. 介入角色定位

根据教师对游戏介入程度不同，可将教师的角色分为

● 非支持性角色

不参与者、导演者。

● 支持性角色

旁观者、舞台管理者、共同游戏者、游戏带头人。教师应根据游戏性质和观察到的情况，不断变换所扮演的角色，推动游戏发展。

2. 介入时机

● 幼儿游戏出现困难时

● 必要的游戏秩序受到威胁时

● 儿童对游戏失去兴趣或准备放弃时

● 游戏内容发展或技能方面发生困难时

3. 介入方式

● 外部干预：教师不直接参与游戏 

● 内部干预：直接参与游戏

4. 介入注意事项

● 分层次指导 

● 慎扮“现实代言人”角色         

● 及时退出
