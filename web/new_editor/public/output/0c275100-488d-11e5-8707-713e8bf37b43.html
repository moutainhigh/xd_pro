第五节 教师心理
考点
2：教师的心理特征

认知特征

在知识结构上

要求教师需要有专业学科内容知识、教育教学、心理学的知识和实践性知识。

在教学能力上

要求教师需要有组织和运用教材的能力、言语表达能力、组织教学的能力、对学生学习困难的诊治能力、教学媒体使用能力、教学机智等。

人格特征

指教师需要有热心和同情心，教师要富于激励和想象的倾向性。

行为特征

一般包括教师教学行为的明确性、多样性、启发性、参与性、任务取向性和及时教学效果评估及其对学生产生的期望效应。

教学效能感

1. 概念

指教师对自己影响学生学习行为和学习成绩的能力的主观判断。

2. 分类

 ●  个人教学效能感指教师认为自己能够有效指导学生，相信自己可以教好学生。

 ●  一般教育效能感是指教师对教育在学生发展中作用等问题的一般看法与判断。

3. 对教师与学生的影响与作用

 ●  教学效能感会对教师行为产生影响：会影响教师在工作中的努力程度，在工作中的经验总结和进一步的学习和教师在工作中的情绪等。

 ●   教学效能感会对学生学业成就产生影响：教师教学效能感与学生的学业成就具有显著的正相关。

4. 影响因素

 ● 外部因素：指社会文化背景、学校的特点、人际关系。

 ● 内部因素：指教师的价值和自我概念。

罗森塔尔效应

罗森塔尔效应又称为皮革马利翁效应、期待效应。指教师的期望或明或暗的传送给学生，学生会按照教师所期望的方向来塑造自己的行为。
