第五节  幼儿园突发事件的处理

考点
1：火灾、地震的应急处理

火灾

1. 拨打“119”电话报警，同时报告主管领导。

2. 立即切断着火楼电源。

3. 按照平时消防演练逃生路线迅速疏散幼儿。

4. 由受过训练的教职工进行三分钟火灾紧急扑灭。

5. 如有人受伤，及时送医，并通知家属。

6. 灭火后，立即聘请律师，配合幼儿园进行相应调查。

7. 配合消防部门调查事故，协助处理善后事宜。

地震

1. 发生后，沉着冷静，看清自己所处位置，组织幼儿有序按照平时熟悉路线逃生，迅速撤离。

2. 震后如被困，及时躲到承重墙之间最小的房间或桌柜下，保护头部。

3. 如幼儿正在睡觉，叫醒幼儿有序躲在床底下或墙角处。

4. 如被建筑物挤压，发出求救信号，等待救援。

5. 如在室外活动，将幼儿集中到操场空旷地带，从精神上抚慰幼儿。
