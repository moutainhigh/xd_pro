第五节 中学德育培养
考点
2：中学德育的途径和方法

途径

1. 思想政治课与其他学科教学，它是德育的基本途径。

2. 社会实践活动

3. 课外、校外活动

4. 共青团组织的活动

5. 校会、班会、周会、时事政策等学习

6. 班主任工作

方法

说理教育法

含义：通过摆事实、讲道理，使学生提高认知、形成正确观念的方法。

地位：是德育工作的基本方法。

实施方法：言语说服和事实说服。

榜样示范法

含义：用榜样人物的高尚思想、模范行为、优异成就来影响学生的思想、情感和行为的方法。

实施方法：

A.选好学习的榜样。  

B.激起学生对榜样的倾慕之情。

C.引导学生用榜样来调节行为，提高修养。

陶冶教育法

含义：教师利用环境和自身的教育因素，对学生进行潜移默化的熏陶和感染，使其在耳濡目染中受到感化的方法。

实施方法：

A 创设良好的情境  

B 与启发疏导相结合  

C 引导学生参与情境的创设

实践锻炼法

含义：让学生参加各种实际活动，在活动中锻炼思想、增长才干、培养优良思想和行为习惯。

实施方法：

A 坚持严格要求  

B 调动学生的主动性  

C 注意检查和坚持

品德修养指导法

含义：教师指导学生自觉主动地进行学习、自我反省，以实现思想转化及行为控制。

实施方法：对学生的自觉意识和自制力提出较高要求，适合高年级学生使用。

品德评价法

含义：通过对学生品德进行肯定或否定的评价而予以激励或抑制，促使其品德健康形成和发展。包括：奖励、惩罚、评比和操行评定。

实施方法：以表扬、奖励为主，批评、惩罚为辅。

依据

1. 德育目标

德育目标是德育工作的预期结果，而德育方法是为实现德育目标服务的。

2. 德育内容

德育内容对德育方法有特定要求。不同性质、不同类别的德育内容需要采用不同性质、不同类别的德育方法。

3. 中学生的年龄特点和个性差异

德育方法的选择和运用必须符合学生的特点，如学生身心发展的年龄特征和个性差异以及品德发展的状况等。
