第二节 小学生学习动机
考点
1：学习动机概述

构成

学习需要

指个体感到某种欠缺而求获得满足的心理状态，即内驱力，包括认知、自我提高、和附属内驱力。

诱因

指能激起定向行为并满足需要的外部条件，包括积极和消极诱因。

功能

1. 激发功能

引发学习内驱力，唤起内部激动状态。

2. 指向功能

使学习行为在初始状态就指向一定的学习目标。

3. 维持功能

使学生在学习过程中，集中注意力，克服不良影响。

4. 调节功能

调节学习行为的强度、时间和方向。

与学习效率的关系

● 动机强度与工作效率是倒U型曲线关系，中等强度的动机最有利于任务的完成。

● 耶克斯和多德森研究表明：在学习较复杂的问题时，动机强度的最佳水平点会低些；在学习任务比较简单时，动机强度最佳水平点会高些。
