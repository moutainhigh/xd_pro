第一节 幼儿园环境创设
考点
2：幼儿园环境创设的原则

一般原则

1.  环境创设的目标要与幼儿园教育目标一致。

2. 环境创设要符合幼儿的年龄特点及身心健康发展的需要。

3. 幼儿与教师共同合作、共同参与，创设环境。

4. 有机结合园内园外环境，形成开放的幼儿教育系统。

5. 确保物质与精神环境的安全。

6. 考虑幼儿园具体情况，因地制宜、勤俭办园。

建筑设计原则

1. 安全原则

园舍建筑牢固安全，尽量建平房，设备应达安全标准。

2. 适应需要原则

符合幼儿身心发展的特点和教育管理的需要。

3. 经济原则

有效利用建筑材料；充分利用建筑物促进幼儿发展。

4. 卫生原则

各种卫生条件应达标。

5. 美观原则

幼儿用房别致多样，办公用房朴实大方，总体布局对称协调。

6. 舒适原则

给予幼儿温度、视觉、听觉和空间的舒适。

室内布置原则

1. 教育性原则

根据当前教育目标、本班幼儿年龄特点和知识经验水平布置环境。

2. 主体性原则

室内布置应有利于幼儿自主自由地活动，让幼儿成为环境的主人。

3. 创造性原则

教师指导幼儿利用想象力，创造性地布置环境。

4. 美观经济原则

教师应勤动手动脑，运用艺术性的方法布置协调相称、结构统一的环境。

室外布置原则

安全性原则

保证设备质量；

全面考虑设备布局，避免潜在危险。

美化、绿化原则

室外场地应使儿童既能享受到自然田野的生活，又能受到美的熏陶。

幼儿园环境创设存在的问题

1. 整体问题：“四少一多”

操作材料少，自主选择少，自由交往少，环境互动少，纪律约束多。

2. 目标偏离

教师根据个人方便与喜好创设环境，偏离目标。

3. 内容不系统

环境内容的组织和呈现缺乏系统的内在联系，未能突出教育意图。

4. 创设环境的观念存在偏差

教师易混淆“孩子参与”与“师生合作”，“家长参与”与“家园合作”，“杂乱”与“丰富”、“空洞”及“有趣”等概念之间的区别。
