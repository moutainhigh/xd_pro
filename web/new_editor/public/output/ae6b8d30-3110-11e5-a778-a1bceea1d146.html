第五节 认知
考点
3：幼儿记忆和想象的发展

记忆

1. 识记（信息输入）

人识别和记住事物。

2. 保持（信息存储）

将编码过的信息以一定的方式存储在头脑中的过程。

3. 再认（信息提取）

经验过的事物再度出现时，能把它认出来。

4. 再现（信息输出）

经验过的事物不在眼前时，能把它重新回想起来。

分类

1. 内容不同

形象记忆、运动记忆

情绪记忆、逻辑记忆

2. 保持时间不同

瞬时记忆：信息保持时间约为1秒钟

短时记忆：信息保持时间短于1分钟

长时记忆：信息保持时间超过1分钟，乃至一生

3. 信息加工和存储内容不同

陈述性记忆：以陈述性知识为内容，即事实类信息。

程序性记忆：又称技能记忆，即对程序性知识进行记忆。

特点

1. 记忆保持时间延长（记忆的潜伏期）

2. 无意识记占优势，有意识记逐渐发展

3. 较多运用机械识记，意义识记开始发展

4. 形象识记占优势，语词记忆逐渐发展

5. 幼儿记忆的意识性和记忆策略逐渐发展

培养

1. 提供形象、鲜明、生动、富有浓厚情绪色彩的识记材料。

2. 向幼儿提出具体、明确的记忆任务。

3. 丰富幼儿的生活经验。

4. 教给幼儿运用记忆策略。

想象

1. 无意想象

（不随意想象）

一种最简单、触及形式的想象。

2. 有意想象（随意想象）

包括再造想象和创造想象。

（1）再造想象

根据言语的描述或图形符号，在人脑中形成相应事物新形象的过程。

（2）创造想象

根据一定目的和任务，不依据现实的描述，在人脑中独立地创造出某种新形象和心理过程。

特点

1. 以无意想象为主，有意想象开始发展

2. 再造想象为主，创造想象开始发展

3. 想象有时和现实混淆

策略

丰富幼儿的表象，在文学艺术和游戏中，创造幼儿想象发展的条件，鼓励和引导幼儿大胆想象，进行适当的训练。
