第四节 学习策略
考点
1：学习策略的概述及分类

含义

学习策略是指学习者有目的的制定的、为了提高学习效果和效率的、复杂的方案。策略是相对学习效果和效率而言的。

特征

学习策略的特征包括主动性、有效性、过程性、程序性。

分类

认知策略

认知策略的分类

(1) 复述策略，它是指在工作记忆中为保持信息，运用内部语言在大脑中重现学习材料或刺激，以便将注意力维持在学习材料上的方法。常用方法有：利用随意记忆和有意识记忆、 排除抑制干扰、整体记忆与分段记忆、多种感官参与和画线、圈点批注。

(2) 精细加工策略：是一种将新学材料与头脑中已有知识联系起来从而增加新信息意义的深层加工策略，是理解性记忆策略。

   常用的方法有：

    ●  记忆术：包括位置记忆法；减缩和编歌诀；谐音联想法；关键词法；视觉联想法；

    ●  做笔记： 教师要督促做笔记，策略：讲的慢一点；重复复杂的材料；呈现做笔记的线索；写出主要信息；提供辅助手段；

    ●  提问；

    ●  生成性学习：指训练学生对他们所阅读的东西产生一个类比或表象，如图形、图像、表格和图解等，以加强其深层理解；

    ●  利用背景知识，联系实际： 精细加工强调在新学信息和已有知识之间建立联系，背景知识的多少在学习中是非常重要的。

(3) 组织策略，指是整合所学新知识之间、旧知识之间的内在联系，形成新的知识结构。常用策略包括列提纲、利用图形和利用表格。

元认知策略 

指学生对自己认知过程的认知策略。分为计划策略、监视策略和调节策略。


   ●  计划策略：指设置学习目标、浏览阅读材料、产生待回答问题以及分析如何完成学习任务。

   ●  监控策略：包括元认知监视和自我提问法；元认知监视是根据认知目标及时评价反馈并正确估计。自我提问法是指需要制定“问题单”；训练学生监控思维过程。

   ●  调节策略是根据对认知活动结果的检查，及时调整。 

古代小学教育

  1. 等级性
  2. 为统治阶级服务
  3. 对儿童进行管制与灌输，采用刻板的识记、背诵教育方式。

近代小学教育

  1. 注重培养学生身心健康
  2. 教育方法多样化
  3. 内容世俗化，课程设置更加科学
  4. 对象普及化，女孩获得教育权利
  5. 教育体系科学化，幼稚园纳入教育体系

当代小学教育

  1. 教育目的注重以人为本，以培养全面发展的人为核心，学科内容的设置和教学方法更加注重儿童的身心发展
  2. 教育对象更为普及，义务教育在许多地方得到全面实施
  3. 重视教师发展，教师制度逐渐规范化
  4. 教育实验涉及的方面更广泛
