第四节 身体与动作
考点
1：幼儿身体发育的规律与特点

幼儿身体发展的规律

1. 连续性和阶段性的统一

幼儿身体发育是连续性和阶段性的统一。幼儿发育从幼稚到成熟是个连续的过程，不是间歇式、跳跃式的过程。该过程可分为若干阶段，这些阶段之间相互联系，如果前面阶段出了问题，就会影响后面阶段的发育。

2. 速度为波浪式

幼儿身体发育的速度是波浪式的。幼儿身体发育的速度个不是匀速的，也不是加速度的，而是有快有慢，呈波浪式的。生长发育的第一个突增期在两岁以前，第二个突增期在青春期。

3. 程序性

（1）头尾律

出生以后，头颅继续快速发育，然后是躯干，最后才是四肢。

（2）正侧律

从人体中部到人体边缘的发展。

4. 系统发育不均衡又统一协调

身体各系统的生长发育存在四种模式。

（1）神经系统的发育在胎儿时期和出生后一直是领先的。

（2）淋巴系统的发育趋势是最初几年发育较快，后又逐渐萎缩至成人水平。

（3）一般系统的发育存在两个发育的突增期。

（4）幼儿期生殖系统基本没有发育。

5. 个体差异

幼儿生长发育具有个体差异性。尽管每个幼儿在发育过程中，都存在上述规律，但由于遗传和环境的不同，每个人又都存在着胖瘦、高矮、智愚、强弱等方面的差异。

幼儿身体发育的特点

1. 身长中心点下移

随着年龄的增长，下肢增长的速度加快，身长的中点逐渐下移。

2. 体围发育由上而下，由中心而末梢

（1）体围通常由头围、胸围、腰围、臀围等指标组成。

（2）婴幼儿身体发育的顺序是由上而下，由中心而末梢。头部最先发育，然后是躯干、上肢，最后才是下肢，但无论是骨骼、肌肉、血管和神经，都是按先中心后末梢的顺序进行的。

3. 器官系统发育

婴幼儿各器官系统的发育不平衡，有先后快慢的差别。神经系统最先发育成熟，而生殖系统到儿童期末才加快发育。身高和体重有两个高峰期，第一个在出生后的一两年，第二个在青春发育期。
