第六节 心理学的基础知识
考点
3：意志过程

概念

意志是有意识地确立目的，调节和支配行动，并通过克服困难和挫折，实现预定目的的心理过程。

特征

• 意志有明确目的性。

• 意志对活动有调节作用。

• 克服困难是意志最重要的特征。

• 意志以随意动作为基础。

意志行动的基本阶段

1. 采取决定阶段

（1） 动机冲突

● 双趋冲突

同时并存两种能满足需要的目标，它们具有相等的 吸引力，但只能选择其中之一是产生的动机冲突。

● 双避冲突

同时遇到两个力图回避的威胁性目标，但只能避其中之一所产生的冲突。

● 趋避冲突

同一目标既有吸引力，又有排斥力，人既希望接近，同时又不得不回避，从而产生的冲突。

● 多重趋避冲突

对含有吸引力与排斥两种力量的多种目标予以选择时所发生的冲突。

（2） 确定行动目的

这是意志活动最重要的环节。

（3）选择方法，制定计划

为了实现目的，必须选择正确行动方法和制定恰当计划。

2. 执行决定阶段

（1）执行决定是一个不断克服困难的过程。

（2）执行决定要接受成败的考验。

品质与相反品质

意志的自觉性

意志的自觉性，是指对行动目的有深刻认识，能自觉支配自己行动。与其相反的意志品质是易受暗示性与独断性。

意志的果断性

意志的果断性，是指一个人能否善于明辨是非，迅速而合理采取决定和执行的品质，与此相反的意志品质是优柔寡断和草率。

意志的坚持性

意志的坚持性，是指能否坚持决定，克服困难和障碍，完成既定目的方面的品质，与此相反的意志品质是顽固执拗和见异思迁。

意志的自制性

意志的自制性，是指能否善于控制和支配自己行动方面的意志品质。与此相反的意志品质是任性和怯懦。
