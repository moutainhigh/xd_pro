第三节教师观
考点
5：教师的主观能动性

含义

主观能动性亦称“自觉能动性”。

指人的主观意识和实践活动对于客观世界的反作用或能动作用。 

教师主观能动性的内容

出工

即指一个工作人员在社会或单位规定的带有强制性的工作时间段上班，通常为8小时。  

出力

即指一个工作人员在工作时间内的精力投入程度。 

提高教师主观能动性的方法

1. 增强教师对一般的教学理念的认同意识。

2. 检查教师的教案，对教案进行互相学习和观摩。

3. 建立起一套科学的考核评价标准体系。

4. 形成科学有效的考核操作机制，奖惩分明。

5. 应当设立一套独立的机构，组建一支训练有素、尽职尽责的质量考评和监管队伍。

6. 加强激励机制的建设和加大奖惩力度，鼓励教师不断创新。
