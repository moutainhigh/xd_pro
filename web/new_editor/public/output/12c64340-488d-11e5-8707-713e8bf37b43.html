第五节 教师心理
考点
3：教师成长心理

教师的成长历程

第一阶段：关注生存阶段

 ●  一般是新教师，他们非常关注自己的生存适应性.

 ●  有些新教师可能会把大量的时间都花在如何与学生搞好个人关系上。

第二阶段： 关注情景阶段

在此阶段教师关心的是如何教好每一堂课的内容，一般总是关心诸如班级的大小、时间的压力和备课材料是否充分等与教学情景有关的问题。

第三阶段：关注学生阶段

教师将考虑学生的个别差异，能否自觉关注学生是衡量一个教师是否成长成熟的重要标志之一。

教师获得成长的基本办法

古代小学教育

  1. 等级性
  2. 为统治阶级服务
  3. 对儿童进行管制与灌输，采用刻板的识记、背诵教育方式。

近代小学教育

  1. 注重培养学生身心健康
  2. 教育方法多样化
  3. 内容世俗化，课程设置更加科学
  4. 对象普及化，女孩获得教育权利
  5. 教育体系科学化，幼稚园纳入教育体系

当代小学教育

  1. 教育目的注重以人为本，以培养全面发展的人为核心，学科内容的设置和教学方法更加注重儿童的身心发展
  2. 教育对象更为普及，义务教育在许多地方得到全面实施
  3. 重视教师发展，教师制度逐渐规范化
  4. 教育实验涉及的方面更广泛
