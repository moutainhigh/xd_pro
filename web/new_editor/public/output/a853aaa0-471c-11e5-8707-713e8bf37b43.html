第三节 幼儿园健康教育
考点
2：身体保健教育

内容

1. 生活习惯和能力

包括盥洗的有关知识、方法和技能，穿脱衣服的有关知识和技能，保护个人和周围环境清洁卫生的有关知识、技能及情感态度，生活作息的有关知识和习惯，饮食卫生的习惯等。

2. 饮食与营养

包括饮食的有关知识和技能，常见食物的名称及营养知识，营养与健康的关系，膳食平衡的有关知识等。

3. 人体认识与保护

包括身体的主要器官及其主要功能，保护器官的主要知识及技能，预防接种的有关知识，常见疾病的预防与治疗常见外伤的简单处理知识及方法，心理健康的有关知识等。

4. 保护自身安全

保护自身安全的主要措施及相关知识。

方法

1. 动作和行为练习法

反复练习已学过的行为技能，加深理解，使之形成稳定下来。

2. 讲解演示法

教师边讲解边结合动作、实物或模型演示，具体形象地向幼儿传授有关健康的知识和技能。

3. 情景表演法

教师现场通过录像向幼儿展示生活情景，让幼儿观察和分析情景中所涉及的健康问题。

4. 讨论评议法

教师鼓励幼儿提出问题，发表意见，最后得出结论，达成共识。

5. 感知体验法

让幼儿通过各种感官来认识和判别事物特性。
