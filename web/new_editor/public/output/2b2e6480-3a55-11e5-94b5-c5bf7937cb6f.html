第一节 小学语文课程标准
考点
2：语文课程目标

总目标

1. 培养爱国主义、集体主义、社会主义思想道德和健康的审美情趣，发展个性，培养创新精神。

2. 认识和汲取中华文化。

3. 养成良好的学习习惯，掌握学习语文的方法。

4. 发展思维，学习方法，养成实事求是的态度。

5. 主动探究，激发想象力和创造潜能，在实践中学习和运用语文。

6. 学会汉语拼音。

7. 能独立阅读，运用多种阅读方法。

8. 能表达自己的见闻、体验和想法。

9. 运用口头语言进行人际沟通和社会交往。

10. 学会使用工具书。

阶段性目标

第一阶段（1-2年级）

识字与写字

1. 认识常用汉字1600个左右，其中会写800个左右。

2. 掌握汉字的基本笔画和常用的偏旁部首,能按笔顺规则写字。

3. 努力养成良好的写字习惯。

4. 学会汉语拼音。

5. 学会独立写字。

 阅读

1.  用普通话朗读课文。

2. 了解课文中词语的意思，积累词语。

3. 阅读童话、寓言、故事。

4. 诵读儿歌、儿童诗和古诗。

5. 认识常用的标点符号。

6. 积累成语和名言警句。

 口语交际

1.  学说普通话并养成习惯。

2. 听别人讲话并了解讲话内容。

3. 听故事、看影像并能复述情节。

4. 能讲述小故事和自己的见闻。

5. 与别人交谈，态度自然大方，有自信心。

综合性学习

1. 有好奇心，能提出问题，共同讨论。

2. 观察大自然，用口头或图文等方式表达自己的见闻和想法。

第二阶段（3-4年级）

识字与写字

1. 认识常用汉字2500个左右，其中1600个汉字会写。

2. 有独立识字能力会运用音序检字法和部首检字法查字典、词典。

3. 能书写正楷字。

4.  写字姿势正确，有良好的书写习惯。

 阅读

1.  用普通话朗读课文。

2. 学会默读。

3. 理解和体会词句的意思和作用。

4. 把握主要内容，体会思想感情。

5.  能复述大意，感受作品，并与他人交流。

6. 诵读优秀诗文。

7. 体会和了解句号与逗号、冒号与引号的一般用法。

8. 积累优美词句、精彩片段以及语言材料。

9. 养成读书看报的习惯，收藏图书资料。课外阅读量不少于40万字。

习作

1. 乐于书面表达，增强自信心。

2. 观察周围世界，写下自己的见闻感受、想象及好的文章和作品。

3. 能用简短的书信及便条进行交流。

4.  积累语言材料。

5. 学习修改习作中明显的错误。

6. 课内习作每学年16次左右。

口语交际

1.  能用普通话进行交谈。

2. 听人说话能把握主要内容，并能进行简单转述。

3. 能清楚地讲述见闻，并说出你自己的感受和想法。

 综合性学习

1. 能提出学习和生活中的问题，共同讨论。

2. 观察大自然，用书面表达自己的观察所得。

3. 学会学习和合作。

4. 能尝试用语文知识和能力解决简单问题。

第三阶段（5-6年级）

识字与写字

1. 累计认识常用汉字3000个左右，其中2500个左右会写。

2. 硬笔书写开顺，行款整齐，力求美观，有一定的速度。

3. 能用毛笔书写楷书。

4. 写字姿势正确，有良好的书写习惯。

阅读

1. 能用普通话朗读课文。

2. 默读有一定速度。

3. 辨别词语的感情色彩。

4. 体会作者的思想感情，领悟文章基本的表达方法。

5.  描述自己印象最深的场景、人物、细节，说出感受。

6. 体会顿号与逗号、分号与句号的不同用法。

7.  扩展阅读量。

习作

1. 习作是为了自我表达和与人交流。

2. 留心观察周围事件，丰富自己的见闻，珍视个人的感受，积累素材。

3. 能写作文。

4. 修改习作，并主动与人交换修改。

5. 要有一定的速度。

口语交际

1. 尊重和理解对方。

2. 乐于讨论，敢于发表意见。

3. 听人说话认真、耐心，能抓住重点，并能简要转述。

4. 表达要有条理，语气、语调适当。

5. 能做发言。

6. 注意语言美。

综合性学习

1. 利用信息渠道获取资料，尝试做研究报告。

2. 策划校园活动和社会活动，写活动计划和活动总结。

3. 组织讨论、专题演讲课题。

4. 了解查找运用资料的方法。
