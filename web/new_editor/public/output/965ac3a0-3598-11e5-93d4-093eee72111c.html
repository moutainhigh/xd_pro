第七节 幼儿园艺术教育
考点
2：音乐教育

特点

1. 形式的集体性

幼儿园音乐教育的主要组织形式是集体活动。

2. 内容的综合性

幼儿音乐活动具有歌、舞、乐一体，创造、表演、欣赏一体的特点。

3. 程序的层次性

幼儿园音乐教育要遵循循序渐进的原则。

4. 材料使用的灵活性

幼儿音乐教育使用材料包括基本材料和辅助材料，材料使用有较高的灵活性。

内容

1. 歌唱活动

包括歌曲、歌唱的表演形式、歌唱的简单知识和技能。

2. 韵律活动

是指在音乐伴奏下所进行的以身体运动和身体造型为表达媒介的一种艺术活动，包括韵律动作及其组合、韵律活动的表演形式、韵律活动的简单知识和技能、韵律活动的常规。

3. 打击乐器演奏活动

指在音乐声中有节奏地敲打某些打击乐器的活动，是培养幼儿节奏感的重要途径。

4. 音乐欣赏活动

包括倾听周围环境的声音、欣赏音乐作品及音乐欣赏的简单知识和技能。

方法

1. 示范法

教师通过演唱、演奏和动作表演，使幼儿直接感受，获得清晰的音乐表象提高学习兴趣。示范时注意教师的位置，使每个幼儿都能清楚地观察、感知，一般示范三次左右。

2. 听唱教学法

包括整体教唱和分句教唱。
