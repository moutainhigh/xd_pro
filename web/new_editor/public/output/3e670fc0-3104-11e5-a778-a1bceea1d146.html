第七节 《幼儿园教育指导...
考点
1：《幼儿园教育指导纲要》

基本观点

1. 城乡各类幼儿园都应从实际出发，因地制宜地实施素质教育。

2. 幼儿园应与家庭、社区密切合作，与小学相互衔接，综合利用各种教育资源。

3. 幼儿园应为幼儿提供健康、丰富的生活和活动环境，满足其发展需要。

4. 幼儿园教育应尊重幼儿的人格和权利，尊重幼儿身心发展的规律和学习特点，以游戏为基本活动，保教并重，关注个别差异，促进幼儿个性发展。

要求

健康

1. 建立良好的师生、同伴关系，形成安全感、信赖感。

2. 与家长配合，根据幼儿的需要建立科学的生活常规。

3. 教育幼儿爱清洁、讲卫生，注意保持个人和生活场所的卫生。

4. 密切结合幼儿生活进行安全、营养和保健教育。

5. 开展丰富的户外游戏和体育活动，培养幼儿兴趣，增强体质，提高对环境适应能力。

6. 发展幼儿基本动作，提高动作协调性、灵活性。

7. 在体育活动中，培养幼儿坚强、勇敢、不怕困难的意志品质和主动、乐观、合作的态度。

语言

1. 创造自由、宽松的语言交往环境，支持、鼓励、吸引幼儿与教师、同伴或其他人交谈。

2. 养成幼儿注意倾听的习惯，发展语言理解能力。

3. 鼓励幼儿大胆、清楚地表达自己的想法和感受。

4. 引导幼儿接触优秀的儿童文学作品

5. 培养幼儿对生活中常见的简单标记和文字符号的兴趣。

6. 利用图书、绘画等方式、引发幼儿对书籍、阅读和书写的兴趣。

7. 提供普通话的语言环境，帮助幼儿熟悉、听懂并学说普通话，少数民族地区帮助幼儿学习本族语言。

社会

1. 引导幼儿参加各种集体活动，体验交往的乐趣。

2. 为每个幼儿提供表现自己长处和获得成功的机会及自由活动的机会。

3. 在共同活动中，引导幼儿学习自律和尊重他人。

4. 教育幼儿爱护玩具和其他物品，爱护公物及公共环境。

5. 与家庭、社区合作，引导幼儿了解人们的劳动。

6. 充分利用社会资源，激发幼儿爱家乡、爱祖国的情感。

7. 适当地向幼儿介绍我国各民族和世界其他国家、民族的文化，培养理解、尊重、平等的态度。

科学

1. 引导幼儿对身边常见事物和现象的特点、变化规律产生兴趣。

2. 为幼儿探究活动创造宽松的环境。

3. 提供丰富的可操作的材料。

4. 引导幼儿积极参加小组讨论、探索，培养幼儿合作学习的意识和能力。

5. 引导幼儿对周围环境中的数、量、行、时间和空间产生兴趣，建构初步的数概念。

6. 从生活或媒体中幼儿熟悉的科技成果入手，培养幼儿对科学的兴趣和对科学家的崇敬。

7. 在幼儿生活经验的基础上，培养初步的环保意识和行为。

艺术

1. 引导幼儿接触周围环境和生活中美好的人、事、物，丰富、激发情趣。

2. 在艺术活动中面向全体幼儿，因材施教。

3. 提供自由表现的机会，鼓励、尊重幼儿的创造。

4. 帮助幼儿提高表现的技能和能力。

5. 指导幼儿利用身边物品美化生活或开展其他活动。

6. 为幼儿创设展示自己作品的条件，引导幼儿相互交流、欣赏。
