第二节 儿童发展理论流派
考点
4：认知理论流派

认知结构

1. 图式

是主体动作的认识结构，个体对世界的知觉、理解和思考的方式，即心理活动的结构和组织。

2. 同化

主体将环境中的信息纳入并整合到已有的认知结构的过程。

3. 顺应

当主体图式不能适应客体要求时，改变原有图式或创造新的图式以适应环境需要的过程，是质的变化。

4. 平衡

通过同化和顺应的相互作用达到符合环境的不断发展着的平衡状态，即整个心理的发展过程。

影响因素

1. 成熟：机体的成长。

2. 自然经验：与外界物理环境的接触而获得知识。

3. 社会经验：在社会相互作用和社会传递过程中获得的经验。

4. 平衡化：主客体相互作用，达到平衡过程，使儿童心理不断从低级向高级发展。

儿童认知发展的阶段

阶段：感知运动阶段（0-2岁）

特点：婴儿依靠感官接触外界世界，儿童还没有语言和思维，逐渐形成客体永恒性概念。

阶段：前运算阶段（2-7岁）

特点：儿童能用表象和语言作为中介来描述外部世界，儿童以自我为中心，思维不可逆，不具有守恒概念。

阶段：具体运算阶段（7-11岁）

特点：儿童具有了守恒概念,儿童思维具有可逆性，开始具有逻辑思维。

阶段：形式运算阶段（11岁以后）

特点：儿童具有抽象概括思维，思维具有可逆性、补偿性和灵活性。
