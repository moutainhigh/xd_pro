第五节 教育学的基础知识
考点
2：教育学的发展

萌芽阶段  

1. 中国

作品：《学记》

意义：中国古代、世界古代最早的一篇专门论述教育、教学问题的论著，尤其以教学原则的总结最为突出。。

人物：孔子

作品：《论语》

意义：中国古代最伟大的教育家和教育思想家，主张“有教无类”，重视因材施教。

2. 西方

人物：苏格拉底

贡献：“产婆术”

人物：柏拉图 

贡献：《理想国》“寓学习于游戏”。

人物：亚里士多德

贡献：首次提出了“教育遵循自然”的原则。

人物：昆体良

贡献：《雄辩术原理》是古代西方最早的教育理论著作。

教育学的创立阶段

人物：夸美纽斯

贡献：捷克著名教育家、“教育学之父”，其作品《大教学论》是教育学形成一门独立学科的标志。

观点：

1. 关于教学原则：教育要遵循人的自然发展；

2. 关于教学制度：班级授课制以及教学的原则、方法；

3. 关于教学思想：“泛智教育”广泛地把自然知识传授给普通人的。

人物：卢梭

贡献：法国思想家，代表作《爱弥儿》

观点：倡导自然教育和儿童本位的教育观。

人物：康德

观点：人所有的自然禀赋都有待发展，教育目的就是使每个人都得到自我完善。

人物：裴斯泰洛齐

贡献：瑞士教育家，代表作《林哈德和葛笃德》

观点：

1. 倡导自然主义教育思想；

2. 最早提出“教育心理学化”的主张；

3. 提倡情感教育，爱的教育；

4. 西方教育史上第一位将“教育与生产劳动相结合”这一思想付诸实践的教育家。

人物：洛克

贡献：英国哲学家，代表作《教育漫话》

观点：

1. 提出“白板说”；

2. 主张绅士教育。

人物：赫尔巴特

贡献：《普通教育学》的出版标志着规范教育学的建立。

观点：

1. 他将伦理学和心理学作为教育学的理论基础，奠定了科学教育学的理论基础；

2. 他强调教师的权威作用，强调教师的中心，形成了传统教育教师中心、教材中心、课堂中心的三中心论；

3. 提出了“四阶段教学”理论；

4. 他提出了教育性教学原则。

人物：杜威

贡献：《民主主义与教育》，是20世纪初实用主义教育学的代表人物。

观点：

1. 新三中心论：“儿童中心（学生中心）”“活动中心”“经验中心”；

2. “教育即生活”“教育即生长”“教育即经验的改造”；

3. 主张“在做中学”；

4. 学校即社会；

5. 提出五步教学法。

人物：陶行知

贡献：《中国教育改造》

观点：提出了“生活即教育、社会即学校、教学做合一”三大主张，生活教育理论是其理论核心。

教育学的发展阶段

1. 马克思主义教育学

人物：凯洛夫

作品：《教育学》

意义：总结苏联20世纪20~30年代教育正反两方面经验，论述了全面发展的教育目的，对我国建国初期的教育有着广泛影响。

人物：马卡连柯

作品：《教育诗》《论共产主义教育》

意义：核心是集体主义教育思想。

人物：杨贤江

作品：《新教育大纲》

意义：我国第一部以马克思主义为指导的教育学著作。

2. 现代教育理论

人物：赞可夫

作品：《教学与发展》

意义：理论核心是“以最好的教学效果使学生达到最理想的发展水平”，提出了发展性教学理论的五条教学原则，即高难度、高速度、理论知识起主导作用、理解学习过程、使学生包括差生都得到一般发展的原则。

人物：布鲁纳

作品：《教学过程》

意义：提出了结构主义教学理论；倡导发现教学法。

人物：瓦根舍因

作品：《范例教学原理》

意义：创立了范例教学理论。

人物：苏霍姆林斯基

作品：《给教师的一百条建议》《把整个心灵献给孩子》《帕夫雷什中学》

意义：全面和谐教育思想，其著作被称为“活的教育学”。

人物：布鲁姆

作品：《教育目标分类学》

意义：提出了掌握学习理论。
