第一节 阅读理解能力
考点
3：概括归纳能力

第一步：分析结构

记叙文

1. 记事文

从事情发展的先后时间、地点转换、阶段入手分析。

2. 记人文

从人物成长的阶段、人物所在地点转换、人物情感变化入手分析。

3. 写景状物文

从观察景物的地点变换、时间变化入手分析。

议论文

1. 注意文章是以立论为主，还是驳论为主。

2. 注意提出问题、分析问题、解决问题的层次和逻辑关系。

说明文

1. 说明事理和事物结构的文章，一般按照人们的认识规律和观察顺序安排结构。

2. 介绍前沿科技类的，大多按照“发明—实验—结果”的顺序安排结构。

3. 揭示事物发展过程的，一般按照事物发展进程安排结构。

第二步：把握思路

1. 分清层次

包括分清全文的层次和某一部分的层次。

2. 摸清关系

主要是开头和结尾与主体部分的关系。

3. 抓题目

题目或表面写作对象，或揭示文章线索，或隐含写作顺序。

4. 抓中心句

中心句往往出现在段首或段尾，抓住中心句就抓住了段落核心。

第三步：归纳要点

1. 归纳途径

是找出相关的概括性语句和提取精要，独立归纳。

2. 需要注意

尽量用原文词语作归纳，对相关文字作要点的层次分析，防止遗漏。

 第四步：分析、概括观点与态度

1. 从概括性强的句子入手

2. 从文中运用的材料入手

3. 从作者的评述入手

4. 抓关键词
