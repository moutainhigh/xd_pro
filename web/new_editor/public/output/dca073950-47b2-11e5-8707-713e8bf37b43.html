幼儿发展评价方法之观察法
考点
2：皮亚杰认知发展阶段论

古代小学教育

  1. 等级性
  2. 为统治阶级服务
  3. 对儿童进行管制与灌输，采用刻板的识记、背诵教育方式。

近代小学教育

  1. 注重培养学生身心健康
  2. 教育方法多样化
  3. 内容世俗化，课程设置更加科学
  4. 对象普及化，女孩获得教育权利
  5. 教育体系科学化，幼稚园纳入教育体系

当代小学教育

  1. 教育目的注重以人为本，以培养全面发展的人为核心，学科内容的设置和教学方法更加注重儿童的身心发展
  2. 教育对象更为普及，义务教育在许多地方得到全面实施
  3. 重视教师发展，教师制度逐渐规范化
  4. 教育实验涉及的方面更广泛
