第一节 阅读理解能力
考点
1：文字表述解读

理解文中重要概念

1. 重要概念的界定

重要概念指的是那些与文段的整体内容或要传达的主要信息密切相关的概念。包括：

● 与写作对象或文段主旨密切相关的概念

● 在文中被临时赋予特殊含义或深层含义的概念

● 反映具体语境的概念

2. 概念理解的技巧

3. 答题步骤

● 领会字面意义，明确此举难点

● 根据字面意义，确定筛选标准

● 根据筛选标准，划出筛选范围

● 逐一对照比较，初选相关词句

● 根据题目角度，整理答案文字

理解文中重要句子

1. 重要句子的界定

重要句子指的是那些对文意表达起重要作用的关键性语句。包括：

● 能点明主旨或能显示脉络层次的关键性语句；

● 在文中起重要作用的中心句、总结句、过渡句或对文脉的推进与转接有关键作用的语句；

● 内涵较为丰富而且具有提示性或引导性的语句；

● 比较含蓄的有深层含义的语句。

2. 句子理解的技巧

（1）答题技巧

● 语境分析法

● 修辞提示法

● 层次划分法

（2）作答流程

● 巧用信息整体把握

● 确定区域圈点勾画

● 尊重原文摘取信息

● 先拟草稿再写全顺
