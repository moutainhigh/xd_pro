第三节 幼儿园健康教育
考点
1：幼儿园健康教育概述

概念

幼儿园健康教育是根据幼儿心理发展的特点，以提高幼儿的健康认识、改善幼儿的健康态度、培养幼儿的健康行为、维护和促进幼儿的健康行为为目标的教育活动。

特点

1. 时间的延续性

长期的巩固才能养成健康行为。

2. 空间的渗透性

健康教育会渗透到日常生活、游戏及其他的教学活动中。

3. 实施者的多元化

幼儿园中除教师外的其他成人群体也能成为健康教育的实施者。

4. 成效的长期性

健康教育的需要幼儿努力坚持才能取得效果。

目标

1. 健康：身体健康，在集体生活中情绪安定、愉快。

2. 卫生：生活卫生习惯良好，有基本的生活自理能力。

3. 保健：知道必要的安全保健常识，学习保护自己。

4. 动作：喜欢参加体育活动，动作协调、灵活。
