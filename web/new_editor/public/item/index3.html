2：我国小学教育的发展

营造良好的情绪环

幼儿的情绪不稳定，很容易受到周围环境的影响，其情绪的发展主要依靠周围情绪氛围的熏陶。幼儿园教育中应注意保持和谐的气氛，创造有利于幼儿情绪放松的环境，同时建立良好的师幼关系。

成人情绪自控示范

优秀的幼儿教师应将自己的消极情绪留在教室外，调整好自己的情绪状态，以积极饱满的情绪与幼儿互动，才能使幼儿保持良好的情绪状态。

积极鼓励和引导

1. 正面肯定和鼓励。

2. 耐心倾听幼儿说话。

3. 正确运用暗示和强化。

正确理解和应对幼儿的情感反应

1. 对幼儿表现出的所有情绪都保持敏感。

2. 对幼儿正在体验的情绪做出非判断性评价。

古代小学教育

  1. 等级性
  2. 为统治阶级服务
  3. 对儿童进行管制与灌输，采用刻板的识记、背诵教育方式。

近代小学教育

  1. 注重培养学生身心健康
  2. 教育方法多样化
  3. 内容世俗化，课程设置更加科学
  4. 对象普及化，女孩获得教育权利
  5. 教育体系科学化，幼稚园纳入教育体系

当代小学教育

  1. 教育目的注重以人为本，以培养全面发展的人为核心，学科内容的设置和教学方法更加注重儿童的身心发展
  2. 教育对象更为普及，义务教育在许多地方得到全面实施
  3. 重视教师发展，教师制度逐渐规范化
  4. 教育实验涉及的方面更广泛
